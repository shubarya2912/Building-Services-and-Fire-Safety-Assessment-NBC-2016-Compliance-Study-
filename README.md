This project was completed as part of the Building Services (CIL7690) course at IIT Jodhpur.
It focuses on assessing building performance and fire safety measures through:

Analysis of ventilation, heating, cooling, and acoustics systems.

Fire safety compliance mapping based on NBC 2016 Part 4 guidelines.

Field inspection and data compilation using MS Excel.

Development of recommendations for improving emergency preparedness and building safety.

Tools & Techniques: NBC 2016, field inspection, compliance mapping, MS Excel
